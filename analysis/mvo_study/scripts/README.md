# MVO Study Scripts

This folder contains all scripts for the MVO analysis pipeline:
- `config.py`: Paths and analysis parameters
- `data_loader.py`: Data ingestion and filtering
- `estimators.py`: Expected returns and covariance estimators
- `run_analysis.py`: Main analysis workflow
- `plots.py`: Plotting utilities
- `report.py`: PDF report generation

See `run_analysis.py` for the main entry point.
